# DoubanAlumGet (dbablum-get)

豆瓣相册下载器

运行 `dbalum_get album_id -n album_name` 可以下载相册中的图片

```
$ dbalum_get -h
usage: 豆瓣相册下载器 [-h] [-n ALBUM_NAME] album_id

positional arguments:
  album_id       要下载的豆瓣相册ID(例如: 1639309626)

optional arguments:
  -h, --help     show this help message and exit
  -n ALBUM_NAME  相册名称
```
